我在网易游戏

桂载铎

{ 北京大学 | 游戏设计师 }

出于对电子游戏的爱好和向往，我选择了国内最优秀的游戏制作公司，
投入了游戏策划的行业。这一年的工作告诉我，我并没有选错。
游戏设计很苦，虚拟世界的每一寸土地的每一个细节都存在于你的脑海中。
游戏设计很累，从设计到实现有一段很长的路要走。游戏设计也很快乐，
梦境随自己的意志实现不是每个人都有机会体验到的幸福。游戏设计充满了成就，
如同上帝创世界一般的奇幻。如果你知道了这一切，还是坚定地认为
游戏设计正是你所追求的事业，那么来吧，怀揣着梦想，坚持不懈，
在策划岗位上挥斥方遒，架构自己向往的虚拟世界。

江宇源

{ 北京大学 | 游戏设计师 }

在网易能学到东西。如果把入行游戏制作比做一场修行，
那网易游戏无疑就是一个洞天福地，能够给你开挂一般的提升快感。
在网易能干实事。金子总会发光的，而网易则能让金子亮的更快一点。
在网易能认识人。在网易任何时候都不会寂寞无助，因为总有人陪你一起前行。
如果一个人执着于制作最好的游戏、变革游戏行业以及改变未来一代人的命运，
那么加入网易游戏是他实现梦想最有效的途径。
因为对游戏，我们从不游戏。

蔡坤雨

{ 清华大学 | 游戏研发工程师 }

实习过程中，让我感受最深的是，网易游戏所拥有的完备的员工培训体系。
作为游戏研发工程师，在这里，我可以获取到各种与游戏研发相关的教程
以及资料，可以选修与研发相关的课程及讲座，在讲座的现场，
还可以与公司的技术大牛探讨各种技术疑难，寻求解决方案。
对于一个初入游戏行业的新手来说，这套培训体系无疑为个人的职业发展
提供了绿色通道。对于研发工程师来说，这种无障碍的技术交流渠道，
为工程师提供了一种快速解决技术难题的途径。

吴羽飞

{ 清华美院 | 游戏视觉设计师 }

在这里，我切实的感受到了由一大批积极优秀的年轻人聚集在一起所散发出的
正能量。几乎每个人都有自己独特的闪光点，同时，
大家都毫不吝啬的分享自己的经验成果，携手并进。短短的二个月实习我
就交到了不少能推心置腹的好友，让我在异乡的生活也充满着归属感。
实习结束不久，工作接触到的实际项目就已正式上线，进入游戏后看到自己
创作的内容，虽然还很微不足道，但心中的激动与成就感已经溢于言表。
短短的二个月时间，我就已经与网易游戏结下了深厚的羁绊，
未来的道路我也期待着与网易同行，携手掀起一股股游戏行业的浪潮。

董文婷

{ 香港中文大学 | 品牌管理培训生 }

这是一个包容的大家庭，有一群可爱的同事。在这里，你可以最大限度地做自己，
不用担心是否要西装革履，不用害怕表达观点，因为"你"的想法是最重要的。
生活上，加班累了可以去咖啡吧小憩片刻，和同事脑爆聊聊想法，
健身房里也常常留下小伙伴们汗流浃背的身影。工作中，每一位老员工
都是你最好的导师，内部培训和实践机会也始终敞开着大门。对我来说，
实践就是最好的学习，几个项目的历练让我从零经验新人到能够带团队
运作游戏上市，压力与动力并存，但看到成果的那一刻，
真的会觉得自己能从事现在的工作真是太好了。

胡秋扬

{ 浙江大学 | 游戏测试工程师 }

实习过程中，首先要点赞的是一对一导师制。其次，实习生工作的安排
绝不是"打杂"或者"脏活累活"，而是精心准备的项目。团队氛围也给我
留下了深刻印象。小伙伴们，导师们，组里的其他成员们，我们共同组成了一个
大团队。除此之外，公司各种福利迎面而来，和我们撞了个满怀。食堂、咖啡吧、
健身房、游戏厅一应俱全，并且走到哪里都免费。还有不定期掉落的IMAX电影，
漂流出游，组内聚餐唱K，带来种惊喜。实习两个月，我满载而归，收获了
深切友谊，体会到了团队的温馨；收获了满满成就感，体会到了自身技术
和沟通能力的提升。如此美差，一举多得，赶紧来投简历吧！

公司简介

关于网易


网易(NASDAQ:NTES)是中国领先的互联网技术公司，在开发互联网运用、服务及
其他技术方面，网易始终保持着国内业界领先水平。
网易对中国互联网的发展具有强烈的使命感，网易利用最先进的互联网技术，
加强人与人之间信息的交流和共享，实现"网聚人的力量"。
网易成立时间：1997年 (http://www.163.com)
网易游戏成立时间：2001年 (http://game.163.com)
NESDAQ:NTES

关于网易游戏


网易2001年正式成立在线游戏事业部，
与广大游戏热爱者一同成长15年，是全球领先的游戏开发与发行公司。
自主研发了《梦幻西游2》《大话西游2》《天下3》《乱斗西游》《梦幻西游手游》
等几十款倍受玩家喜爱的热门端游和手游，
更独家代理了《魔兽世界》《炉石传说》等多款风靡全球的游戏。
2015年，网易游戏正式开启"游戏热爱者"品牌战略，
用自身行动传导热爱者价值，并全面启动"聚合精品，共享热爱"战略计划，
以全球化视野为游戏热爱者创造和发现好玩的精品游戏，
提高游戏准入门槛，降低用户选择门槛。

员工眼中的网易游戏


网易游戏就像一个大家庭，公司气氛轻松又活泼，最适合年轻人自由心性，不用全套
职业装正襟危坐，大家穿着宽松T恤和夹脚拖鞋来上班，工作就是游戏，
游戏就是工作，既有学术严谨的专业讨论，也有插科打挥的相互调侃，在我们专用
的泡泡群上，即便是部门老大们也不能逃脱被"调戏"
的命运。下班打DOTA再也不用愁找不到队友，身边都是卧虎藏龙的高手。
在网易游戏实习，你可以：
得到业内大牛的专业指导；和国内顶尖的游戏开发达人一起工作；
了解业内领先工作方法；参加业内最核心技术分享，
了解业内领先经验。
