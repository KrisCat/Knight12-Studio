# Venus Alpha（测试中）

使用说明
* [预览网址](http://muwenzi.com)
* 运行环境: node v5.5.0
* Step1: 将public里"img_github"文件夹更名为"img"（图片版权保护，否则无法正常预览）
* Step2: npm intsall
* Step3: npm start
* Step4: 浏览器localhost:8080打开预览
